## Конспект
### 4.1 операторы
* == 	Равенство	 
* != 	Неравенство
* <  	Меньше чем	  
* >  	Больше чем
* <= 	Меньше или равно	 
* >= 	Больше или равно
* & - знак амперсанд
* & - логический оператор И
* | - логический оператор ИЛИ
* Смысл операции || аналогичен операции &&, она так 
же сначала проверяет первое выражение. Но ищет при этом она другое значение.
* ^ исключающее ИЛИ, аналогична |, но возвращает обратное значение
* Т.е. | если два условия true покажет trut 
(или если хотя бы одно верно), а ^ покажет false (либо либо)
* ! - отрицание
* Условие IF (условие) {действие} ELSE {другое действие}
* есть более просты формы записи var c = a != b ? a + b : b;
* до ? условие, после если условие правда, и после : елси улсовие ложь
* Console.BackgroundColor - свойство цвета фона консоли 
(не всего, а того который у наших данных)
* Console.ForegroundColor - свойство цвета шрифта.
