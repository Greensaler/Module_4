## Конспект
### 4.1 операторы
* == 	Равенство	 
* != 	Неравенство
* <  	Меньше чем	  
* >  	Больше чем
* <= 	Меньше или равно	 
* >= 	Больше или равно
* & - знак амперсанд
* & - логический оператор И
* | - логический оператор ИЛИ
* Смысл операции || аналогичен операции &&, она так 
же сначала проверяет первое выражение. Но ищет при этом она другое значен ие.
* ^ исключающее ИЛИ, аналогична |, но возвращает обратное значение
* Т.е. | если два условия true покажет trut 
(или если хотя бы одно верно), а ^ покажет false (либо либо)
* ! - отрицание
* Условие IF (условие) {действие} IFELSE {другое действие} ELSE {действие в остальных случаях}
* есть более просты формы записи var c = a != b ? a + b : b;
* до ? условие, после если условие правда, и после : елси улсовие ложь
* Console.BackgroundColor - свойство цвета фона консоли 
(не всего, а того который у наших данных)
* Console.ForegroundColor - свойство цвета шрифта.

### 4.3 Массивы
* Массив - это набор однотипных данных, как бы много переменных
* Нумерация элементов в массиве начинается с 0
* foreach цикл перебор всех элементов массива
* foreach (var color in favcolors)
{
  switch(color)
}
* В скобках мы указываем переменную, 
в которую будет «помещаться» текущий по порядку элемент массива.
После служебного слова in указывается массив или другая последовательность.
Для переменной необходимо указать тип, и он должен совпадать с типом массива.
* Переменная типа string является массивом литералов
* Много полезных упражений следует потом повторить
